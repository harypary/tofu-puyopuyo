セキュリティ・プライバシーポリシー

1. データセキュリティ

本アプリのゲームデータはデバイス内にのみ保存され、外部サーバーへ送信されることはありません。広告配信のための外部通信はすべてHTTPS暗号化通信で保護されます。氏名・メールアドレス等の個人情報（PII）は一切収集しません。

2. 広告について

本アプリはGoogle AdMobを使用して広告を表示しています。すべての広告リクエストは非パーソナライズ広告（requestNonPersonalizedAdsOnly）として送信しており、ユーザーの行動追跡・プロファイリングは行いません。App Tracking Transparency（ATT）フレームワークは使用していません。

詳細はGoogleの広告に関するポリシーをご確認ください。

3. 収集する情報

本アプリが独自に収集・送信する個人情報はありません。ゲームスコア・設定は端末内にのみ保存されます。
